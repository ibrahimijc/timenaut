# Timewire

## Installation

`git clone git@github.com:Teras23/timewire.git`  
`cd timewire`  
`python3 -m venv venv`  
`source venv/bin/activate`  
`pip3 install wheel`  
`pip3 install -r requirements.txt`  

For python 3.7+  
`cd venv/srv/bootloader`  
`python3 ./waf distclean all`  

## Running for development
`python3 main.py`

## Building for distribution

`python3 setup.py`
